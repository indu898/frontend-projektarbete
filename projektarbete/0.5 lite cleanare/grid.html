Gridkonceptet var påtänkt sedan långt tillbaka redan 2005 fanns en “Advanced layout module”. Under 2010 talet gick de fyra största brower utvecklarna ihop för att utveckla en grid-standard. Grid släpptes och fick stöd av alla dessa browsers år 2017. Grid är det första riktiga tvådimensionella layoutverktyget. Grid systemet bygger till skillnad från flexbox på både kolumner och rader, vilket gör det lättare med komplex layout. Du kan gå från papper eller idé med minimal kod till färdig grundlayout på kort tid. Grid kan användas till hela din layout eller små element på sidan. När du gjort din grid kan du du lätt med flera olika metoder lägga ut ditt material precis där du vill. Det finns inget som säger att du ska använda grid till allt. Grid gör det väldigt enkelt att göra din sida responsiv.
